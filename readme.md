## 시스템 개요

| 항목 | 내용 |
|---|---|
| 버전 | v7.6 |
| 스택 | React 18 JSX (단일 파일) |
| 빌드 도구 | Vite |
| 마지막 업데이트 | 2026-04-22 |
| 검증 견적서 수 | 19건 |

**출력 항목 순서:** 공정합계 → 개발비 → 개당단가 → 총공급가액 → 부가세 → 합계

- **개당단가** = floor(공정합계 ÷ 수량), 개발비·부가세 별도
- **부가세** = 별도 표기 (총공급가액 × 10%)

## 주요 기능

- 박스 규격 입력 → 전개도 치수 자동 계산
- 지종·판형 자동 추천 (총비용 최소화 기준)
- 배치 시각화 (인터로킹 포함, 전개도 패널 오버레이)
- 전개도 미리보기
- 수량별 개당 단가 비교표
- R수(지대 수량) 자동 계산 + 수동 여분 입력 병용
- 지대 단가 직접 입력 / R수 직접 입력 지원

| 조건 | 가로 | 세로 | 회전 |
|---|---|---|---|
| D/H > 0.8 | W + 4D + 14 | 2H + 3D + 1 | 가능 |
| D/H ≤ 0.8 | W + 2D + 14 | H + 2D | 불가 |

**검증:** 315×126×113 → 781×592 ✓ / 315×113×126 → 833×605 ✓

## R수(지대 수량) 계산 공식

### 핵심 원리

```
R = (정미 + 여분) / (500 × 절수)

정미 = ceil(수량 / up수)
절수: 46전지·국전=1 / 하2·4×62·국2=2 / 4×64=4
```

### 자동 여분(Loss) 판단 (19개 견적서 역산 기반)

| 조건 | 여분 |
|---|---|
| 정미 < 500장 (초소량) | **25장** 고정 |
| 기본 | **250장** |
| + 대형 전지 (46전지·국전) | +50장 |
| + UV인쇄 / 형압 / 금박 | +100장 |
| + 합지 (E골·B골) | +350장 (running 제외) |
| + 대량 (정미 > 3,000장) | +정미 × 3% |

**수동 보정:** 여분(Loss) 장수 직접 입력 가능. 입력 시 자동 판단 대신 적용.

### 절수 기준 (연당 장수)

| 판형 | 절수 | 1연(R) |
|---|---|---|
| 46전지, 국전 | 1 | 500장 |
| 하2, 4×62, 국2 | 2 | 1,000장 |
| 하4, 4×64 | 4 | 2,000장 |

### 공정 R (코팅·톰슨)

```
processR = round(정미 / 10) / 10   (0.1R 단위)
```

> 코팅·톰슨은 지대와 별도 공정으로 0.1R 단위 반올림

**검증:** 52×52×93 4up 5000ea → processBase=1.25 → **processR=1.3R**  
코팅 1.3×55,000=71,500 ✓ / 톰슨 1.3×45,000=58,500 ✓

## 배포 가이드

### 요구사항

- Node.js 18+
- npm / yarn

### 설치 및 실행

```bash
# 프로젝트 생성
mkdir cria-quote && cd cria-quote

# package.json 생성
npm init -y
npm install react react-dom
npm install -D vite @vitejs/plugin-react
```

**프로젝트 구조:**

```
cria-quote/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    └── App.jsx        ← quote_system_v7.jsx 내용 + export default
```

**index.html:**

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>코리팩 자동 견적</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
  </body>
</html>
```

**src/main.jsx:**

```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.jsx'

ReactDOM.createRoot(document.getElementById('root')).render(<App />)
```

**App.jsx 마지막 줄에 추가:**

```jsx
export default function App() {
  return <QuoteApp />
}
```

**vite.config.js:**

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({ plugins: [react()] })
```

### 개발 서버

```bash
npm run dev
# http://localhost:5173
```

## 알려진 한계

### 전개도 공식 미확정 케이스

| 케이스 | 상황 | 비고 |
|---|---|---|
| 맞뚜껑 뚜껑 높이 | 현재 `D/2+20` 사용, 실측과 차이 | 4케이스 역산 공식 있으나 H 음수계수로 보류 |
| G형 D/H < 0.2 극단 평판 | 323×234×36 실측 571×507 미재현 | 별도 공식 필요 |
| 접착날개 (glueTab) | 14.3mm 고정 (실측 10~23mm 가변) | 케이스별 수동 조정 필요 |

### 단가 데이터 미확인

- ABL325 4×62 단가 (면적비례 추정값 사용 중)
- 대부분의 지종에서 하4·하2 조합 미확인
- 합지(E골/B골) 판형 완전 미지원

### R수 자동 판단 한계

| 케이스 | 자동 여분 | 실측 여분 | 오차 |
|---|---|---|---|
| ABL295 4000 6up 4×64 | 250장 | 633장 | 이상치 (원인 불명) |
| 틴트 60000 12up | 400장 | 600장 | △ 0.1R |

> 이 경우 **여분 수동 입력** 기능으로 보완

### 합지 자동 감지 미지원
현재 합지(E골/B골) 여부를 자동으로 감지하는 UI 옵션 없음.  
→ 합지 건은 여분 수동 입력에서 **600장** 설정 권장
